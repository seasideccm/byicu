                                                    # 成人重症监护室的电阻抗断层扫描：临床应用和未来方向

                                                    # Jonah Rubina,b 和 Lorenzo Berrab,c

                                                    审查的目的

                                                    电阻抗断层扫描（EIT）是一种新型的、无创的、无辐射的床旁成像和监测工具，可评估和显示肺部通气和灌注的区域分布。尽管它主要是一种研究工具，但迅速出现的数据已开始确定其临床作用，它有望成为重症监护室（ICU）的一个普遍的补充。在这篇综述中，我们总结了支持EIT在成人重症监护室临床应用的数据，强调适当的应用，同时强调未来的方向。

                                                    最近的研究结果

                                                    最近的主要研究主要集中在EIT在设置正确的呼气末梢正压以平衡区域过度扩张和塌陷的作用。在过去的几年里，我们的肺部抢救小组已经证明，将EIT纳入多模式的方法来进行个性化的呼吸机管理可以改善结果，特别是对肥胖者。我们还回顾了最近在COVID期间使用EIT的数据，以及其他广泛的潜在应用。

                                                    摘要

                                                    随着EIT变得越来越普遍，其临床作用也越来越明确，强化治疗师将从对其应用和局限性的清晰理解中受益。

                                                    关键词

                                                    急性呼吸窘迫综合征，重症监护，电阻抗断层扫描，肺部生理学

                                                    # 简介

                                                    电阻抗断层扫描（EIT）是一种新型的、无创的、无辐射的床旁成像和监测工具，可以对区域通气和灌注分布进行逐次可视化和评估，进一步实现床旁呼吸管理的个性化。这一强大的工具提供了原本无法获得的生理学数据，有时可能会避免计算机断层扫描（CT）成像及其运输和辐射风险。最近，研究已经开始界定其在成人重症监护室（ICU）的临床作用，使EIT成为重症监护医生的一个新兴的重要工具。本文为不熟悉复杂的方程式、物理学和技术细节的临床医生回顾了目前EIT的临床应用。正在进行的研究必须专注于利用其独特的特性，以帮助推进和革新ICU管理。

                                                    # 电阻抗断层成像的基本原则

                                                    EIT背后的科学原理以前已经描述过[1,2]。在此，我们简要回顾一下EIT临床整合所需的基本概念。EIT设备包括一条胸带，上面有多达32个电极，与一个控制台和显示器相连（图1a）。电阻率随着呼吸周期中肺泡的伸展和变薄而下降，使EIT能够检测到充气变化（图1b）。

                                                    # 关键点

                                                    • EIT是一种无创的、无辐射的监测工具，用于评估和显示区域通气和灌注情况。- EIT目前主要是一种研究工具，但迅速出现的数据正帮助确定其临床作用。- EIT可能被证明是多模式方法的一个关键组成部分，以实现呼吸机的个性化管理，正如我们的肺部抢救小组已经证明的那样。

                                                    这提供了出色的时间分辨率，但空间分辨率有限。区域分布的精确定位是有限的，但变化的时间可以在每秒50张图像中查看，类似于视频。EIT可以识别区域不均匀性，包括悬雍垂和伴随的过度滞留和无意识，否则会被基于呼吸机的系统指标（即气道压力和潮气量）所掩盖。利用类似的原理，EIT也可以通过停止呼吸循环，同时注入10毫升高渗盐水来检测血流的分布，其意义在本综述的后面部分[3,4]（图1c）。

                                                    与CT相比，EIT测量已被验证为潮气量和通气分布的准确替代物[5]，与氮气冲刷相比，呼气末肺容积的准确替代物[6]。使用各种EIT测量的指数是正在进行的研究的主题，包括 "全球不均匀性指数"，捕捉通气分布的区域异质性[7]，"区域通气延迟"，表征时间异质性[8]，以及 "动态相对区域应变"，将潮气阻抗与呼气末肺阻抗相联系，这与10名COVID患者的病程相关[9]。

                                                    # 识别适当的呼气末正压

                                                    由于EIT可以独特地识别区域过度扩张，许多临床研究已将EIT应用于呼吸衰竭患者的呼气末正压（PEEP）滴定，特别是急性呼吸窘迫综合征（ARDS）。始终存在的挑战是如何最大限度地增加无回流、依赖性肺的招募和通畅性，同时最大限度地减少腹侧、非依赖性肺的过度阻滞，从而避免呼吸机引起的肺损伤和不良血流动力学影响[10,11]。

                                                    Costa等人首次使用EIT来识别最大招募后PEEP递减过程中潮气顺应性的变化（使用EIT衍生的潮气量和呼吸机衍生的驱动压力），认为随着PEEP的增加顺应性的恶化是由于过度滞留，随着PEEP的减少顺应性的恶化是由于塌陷，而理想的PEEP是过度滞留和无意识状态最小的地方[12]（图2）。虽然这假设过度滞留和塌陷同样有害，而且有些人不同意[13]，但这种技术已经在各种ICU人群中得到验证[14,15,16]，并启动了重要的进一步研究。2019年，Costa的PEEP滴定方法与使用静态压力-容量曲线在55名严重ARDS患者中进行了比较。EIT组有更高的PEEP，更好的顺应性，以及2小时后更低的驱动压力，并最终有更好的医院生存率[17]。2021年，一项针对87名中重度ARDS患者的类似研究显示，基于EIT的PEEP方法具有较低的PEEP、较低的驱动压力和较好的生存率[18]。这些研究之间较高和较低的最佳PEEP的差异只是反映了每个样本人群中最佳PEEP的差异，并表明EIT可以在两个方向上确定适当的PEEP。

                                                    图1. 基本的EIT设备和输出。在面板a中，显示了连接到放置在病人胸部的电极带的EIT控制台的图纸。b组显示了输出成像，显示了四个象限的相对区域通气。面板c显示了输出成像，显示了灌注的相对区域分布。EIT，电阻抗断层扫描。

                                                    图2. 用Costa的方法演示PEEP滴定的案例。在a组中，在一个招募动作后，递减的PEEP滴定产生了一个图形描述和表格，在每个给定的PEEP下，肺塌陷的百分比、过度停留和呼吸系统顺应性。根据Costa的方法，最佳的PEEP大约是1012cmH2O，在这里，过度停滞和塌陷的曲线重叠，顺应性最大。在面板b中，在较高的PEEPs（16和14 cmH2O）下的EIT图像显示通气分布没有改善，而驱动压力和呼吸系统的顺应性却恶化了。EIT，电阻抗断层扫描；PEEP，呼气末正压。

                                                    与ARDSNet表格相比，一种使用EIT将PEEP设定在维持呼气末肺容积在10%以内所需的水平的替代方法得到了验证，以改善ARDS患者的驱动压力[19]，以及为严重哮喘患者设定PEEP[20]和为严重ARDS进行体外膜氧合(ECMO)[21]。

                                                    对这两种方法进行了比较，发现总体上在达到的PEEP方面没有明显差异，但在3名患者中存在明显的差异[22]，强调了进一步研究的必要性。

                                                    # 肺部抢救小组和肥胖症

                                                    2014年，马萨诸塞州总医院的肺部抢救小组（LRT）将EIT纳入多模式方法，最终包括食道测压和床边超声心动图，以指导呼吸机管理，特别是对于ARDS肥胖患者的挑战性招募和PEEP滴定，与ARDS相关的未知肺部弹性和肥胖导致的胸膜压力增加相结合，使适当的PEEP广泛变化，难以猜测，并且经常需要其他关于过高的应用气道压力。

                                                    在2019年的一项交叉研究中，我们比较了14名严重肥胖的ARDS患者通过ARDSNet表进行的PEEP滴定与单纯食道测压与LRT方法在招募后的递减PEEP试验中确定的最佳呼吸系统顺应性[23]。LRT组的肺部过度扩张和塌陷最少，顺应性更好，氧合度也更好。回顾过去，LRT方法在指导ICU管理方面是安全、高效和有效的[24]，对BMI>35 kg/m2的患者产生较高的PEEP，而不会导致血流动力学不稳定或过度滞留[25]，与BMI>40 kg/m2的患者的生存率提高有关（31% vs. 40%）。在体重指数大于30 kg/m2的患者中，ECMO的需求量减少[27]，28天的死亡率为31%，1年的死亡率为41%，22%）[26]。这些令人信服的数据支持了通气管理个体化的好处，甚至对死亡率的影响，以及EIT在这方面的作用。

                                                    # 呼吸机不同步现象

                                                    2013年，EIT被证明可以检测到肺部损伤患者的垂体，并显示其通过自发努力产生[28]。这种可能的伤害性现象只能通过像EIT这样的技术来检测，这些技术可以识别肺部膨胀随时间的变化，否则会被全局的、呼吸机指导的参数所掩盖。2018年，较高的PEEP策略被证明可以减弱pendelluft[29]。正在进行的研究正在评估基于EIT的pendelluft测量方法[30]。

                                                    呼吸堆积是呼吸机不同步的另一种形式，可以产生有害的大潮气量，在呼吸机波形上经常被忽略，可以通过EIT[31,32,33]发现。鉴于EIT的耗时性以及呼吸堆积发生的方式和时间的可变性，EIT不太可能成为呼吸堆积的筛查工具，但在利用EIT做其他用途时可能会偶然发现。

                                                    # 肺栓塞检测和通气-灌注匹配

                                                    EIT评估灌注分布需要在呼气末暂停时注入约10mL的高渗盐水并进行电阻抗测量。EIT被证明可以检测已知肺栓塞（PE）区域的通气-灌注（VQ）不匹配[34]，促使一项前瞻性观察研究验证了这一发现[35]。对68名患有低氧血症的ICU患者进行了EIT检查，其中11人患有已知的PE。使用最大阻抗的20%来定义通气和灌注，确定了纯分流（无通气的灌注）、死腔通气（无灌注的通气）和有灌注的通气区域。患有弥漫性肺部疾病的患者在区域通气和灌注方面没有缺陷。有血胸的病人在受影响的区域有通气和灌注的缺陷。与其他患者相比，有肺栓塞的患者有更多的死腔通气，更严重的VQ不匹配，以及更少的肺内分流。在这个队列中，死腔通气量为30%的分界线可以识别出PE，其敏感性为91%，特异性为99%。

                                                    值得注意的是，与CT可以检测和定位血管内梗阻不同，EIT不能区分血管内和血管外的灌注障碍原因，用上述方法，在通气量减少的区域，即使灌注不足是由PE引起的，而通气不足是由其他原因引起的，也会忽略灌注不足的情况。EIT确实具有床边服务、无辐射测试和无需静脉注射造影剂的优点。因此，尽管EIT不可能被证明在诊断上等同于PE的CT，但它可能被证明对指导个别不稳定病人的临床决策有足够的帮助。此外，通过降低死腔通气百分比的临界值，以牺牲特异性为代价提高敏感性，EIT可能被证明能够明确地排除PE。在广泛地将EIT应用于PE诊断算法之前，还需要进一步研究。

                                                    最近，EIT被用来量化ARDS患者的VQ不匹配，死亡率与匹配度较差有关[36]。因此，EIT未来可能也会有预后作用。

                                                    # COVID-19的应用

                                                    COVID大流行引发了进一步的EIT研究，包括已经讨论过的应用，其结果适用于COVID患者和可能更广泛的ARDS患者群体。

                                                    # 正呼气末梢压力

                                                    在一系列15名COVID ARDS患者中，平均BMI为30 kg/m2，通过Costa的方法（增加2cmH2O），基于EIT的PEEP选择与ARDSNet PEEP较高的表格相关，而不会引起血流动力学不稳定或气胸[37]。在另一个系列的15名体重指数较低的患者中，通过Costa的方法得出的EIT最佳PEEP水平与ARDSNet表相似，但有3名患者除外[38]。这种方法指导了一名围产期患者的两次连续ECMO运行，该患者幸存下来[39]。这些研究证明了COVID ARDS的可变生理学，表明个体化滴定可能比依赖ARDS-Net表格更合适，并强调了继续研究的必要性，这与前述众多在广义ARDS预COVID中的类似研究结果一致。

                                                    # 肺灌注

                                                    我们小组报告了成功应用EIT诊断并启动PE的治疗，基于同质通气和不平衡的灌注，在一个插管的、俯卧的COVID ARDS患者中，诊断随后在一天后稳定后被CT证实[40]。在资源有限的大流行病中，除了已知的造影剂、辐射和运输临床上不稳定的病人的危害外，运输病人还增加了传染病传播的风险，EIT被证明是一种有价值的、安全的选择。

                                                    # 新的用途

                                                    一项对5名患者的观察性研究使用EIT成功地证明，通过有针对性的侧向定位来滴定PEEP，使通气较少的一侧保持较高的通气量，可以减少区域过度停留和塌陷，改善左右肺通气/通气不对称[41]。另一项对10名未插管的COVID ARDS患者的研究使用EIT测量插管前通过持续气道正压进行单次PEEP降级（从12mmH2O降到6mmH2O）时呼气末肺阻抗的变化。那些在仰卧时PEEP降级时变化小于40%的患者（n = 4）最终未能使用CPAP，被插管，并且没有存活下来，而那些变化大于50%的患者不需要插管，并且存活下来，这表明EIT在预测结果方面有更多的预后潜力，并且可能确定那些可能从早期插管中受益的患者[42]。

                                                    # 可招募性

                                                    有两项研究用EIT证明了俯卧位对仰卧时潮气量分布不均匀的病人的影响[43,44]。改善通气均匀性和肺顺应性是ARDS期间俯卧位有益于死亡率的一个拟议机制，EIT最终可能有助于在尝试前预测谁可能从俯卧位中受益。目前，这个概念还不成熟，如果没有对ARDS患者暂停代偿的平等性进行研究，那么就很难对其进行研究。

                                                    # 呼吸机断气

                                                    EIT已被用于监测自发呼吸试验（SbT）[45]。2019年的一项观察性研究使用EIT监测30分钟的低PEEP（2cm H2O）SBT，以及那些通过后立即拔管的人[46]。通过和未通过SBT的人在潮气量变化或动脉血气方面没有差异，但那些未通过的人表现出潮气阻抗和基线不均匀性指数的明显减少，这表明EIT在预测成功的SBT和作为拔管后监测的辅助手段方面有潜在的作用。

                                                    # 监测静脉体外膜氧合的肺部

                                                    有研究评估了静脉ECMO患者在超保护性呼吸机设置上的通气分布[[15,16](#15]]。可以选择最佳的PEEP来减少塌陷和过度滞留，但目前还没有研究证明这种PEEP的选择对ECMO的结果有任何影响，甚至与其他方法设置的PEEP有明显不同。一项研究表明，代偿的生理益处也适用于ECMO患者[47]。在21名患者中，基于EIT的最佳PEEP中位数与仰卧位相比减少了4cmH2O，60%以上的患者静态顺应性提高了3mL/cmH2O以上。EIT在指导ECMO患者呼吸机管理这一众所周知的领域的研究方面具有巨大潜力。

                                                    # 检测移植后的肺动脉狭窄

                                                    在首次报道的使用EIT进行移植后肺部灌注的情况中，发现左侧移植物的灌注减少和延迟，是经食道超声心动图[48]看不到的严重吻合口狭窄。

                                                    # 检测气道关闭

                                                    2018年，一个小组描述了一个值得关注的问题，即气道关闭是ARDS常见的、经常被遗漏的生理性并发症，通过呼气末憋气测得的自动PEEP往往低于维持气道通畅所需的水平，而且可以用低流量压力-容积环路来检测[49]。另一个小组证明，EIT也可以识别气道关闭[50]。

                                                    # 吸气或断开通气管时的减员情况

                                                    EIT具有出色的时间分辨率，可以检测到肺容积的快速变化，并被用来证明气管内吸痰或呼吸机断开时的容积损失[51]。EIT随后被用来比较开放和封闭式吸痰[52]。

                                                    # 胸腔积液

                                                    EIT还检测了胸腔积液[53]。还有人用EIT来证明胸腔穿刺的生理效应[54]。超声波仍然是识别胸腔积液和评估胸腔穿刺可行性的首选诊断方法，但未来EIT可能会在不明确的病例中发挥作用，指导谁可以从胸腔穿刺中受益，以及何时受益。

                                                    # 气胸

                                                    EIT还可以检测到气胸，甚至小到20毫升[55]。据报道，当EIT在招募动作中使用时，实时检测到了气胸，导致快速干预[56]。虽然超声和胸片检查可以更迅速地发现气胸，但在EIT监测期间进行的演习中实时发现气胸的能力可以使其更安全地传导。

                                                    # 对胸部物理治疗的反应

                                                    在一项研究中，EIT被用来证明高频胸壁振荡对高分泌患者的疗效[57]。

                                                    # 结论

                                                    EIT是一种新型的、无创的、无辐射的床旁成像和监测工具，可以评估和显示动态的逐次呼吸的肺通气和灌注的区域分布。它目前主要用于研究，但最近越来越多地被证明在临床上是有用的，主要是对肥胖等患者进行个性化的PEEP滴定，这些患者可能不太可能被ARDSNet表格或其他标准化的PEEP选择方法所捕获。EIT的更广泛的应用正在积极探索中。随着它的作用越来越清晰，EIT有可能成为重症监护室武器库中一个无处不在的补充。
